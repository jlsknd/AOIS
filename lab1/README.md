# Лабораторная работа №1 "Реализация арифметических операций с числами в различных двоичных кодах"

## Цель работы

Разработать программный комплекс для выполнения арифметических операций с числами в различных двоичных представлениях:

- Прямой, обратный и дополнительный коды для целых чисел
- Стандарт IEEE-754-2008 для чисел с плавающей точкой
- Двоично-десятичный код (8421 BCD)

**Основное требование:** все операции должны быть реализованы без использования встроенных функций преобразования, с хранением чисел в виде массивов битов размером 32.

---

## Теоретическая часть

### Прямой код
В прямом коде старший бит отводится под знак числа (0 - положительное, 1 - отрицательное), остальные биты хранят модуль числа.

### Обратный код
Для положительных чисел совпадает с прямым кодом. Для отрицательных чисел все биты кроме знакового инвертируются.

### Дополнительный код
Для положительных чисел совпадает с прямым кодом. Для отрицательных чисел получается из обратного кода добавлением 1.

### IEEE-754 (32 бита)
Число представляется тремя полями:
- **Знак (1 бит):** 0 - положительное, 1 - отрицательное
- **Экспонента (8 бит):** хранится со смещением 127
- **Мантисса (23 бита):** нормализованная дробная часть

## Описание классов и методов

### 1. Класс `BinaryConverter` (binary_converter.py)

Базовый класс для преобразования между десятичными числами и двоичными представлениями.

| Метод | Назначение | Входные данные | Выходные данные |
|-------|------------|----------------|-----------------|
| `decimal_to_binary_array()` | Преобразует десятичное число в массив битов | `num` - число, `bits` - разрядность, `signed` - со знаком | Список из 0 и 1 |
| `binary_array_to_decimal()` | Преобразует массив битов в десятичное число | `binary_array` - массив битов, `signed` - со знаком | Десятичное число |
| `binary_array_to_string()` | Преобразует массив битов в строку | `binary_array` - массив битов | Строка вида "01010101" |
| `to_direct_code()` | Получает прямой код числа | `num` - число, `bits` - разрядность | Массив битов в прямом коде |
| `to_reverse_code()` | Получает обратный код числа | `num` - число, `bits` - разрядность | Массив битов в обратном коде |
| `to_additional_code()` | Получает дополнительный код числа | `num` - число, `bits` - разрядность | Массив битов в дополнительном коде |
| `additional_to_decimal()` | Преобразует дополнительный код в десятичное число | `binary_array` - массив битов | Десятичное число |

### 2. Класс `FixedPointOperations` (fixed_point.py)

Реализует арифметические операции с целыми числами в различных кодах.

| Метод | Назначение | Входные данные | Выходные данные |
|-------|------------|----------------|-----------------|
| `add_additional()` | Сложение в дополнительном коде | `num1`, `num2` - числа | Словарь с результатом |
| `subtract_additional()` | Вычитание через дополнительный код | `num1`, `num2` - числа | Словарь с результатом |
| `multiply_direct()` | Умножение в прямом коде | `num1`, `num2` - числа | Словарь с результатом |
| `divide_direct()` | Деление в прямом коде | `num1`, `num2`, `precision` - точность | Словарь с результатом |


### 3. Класс `IEEE754` (ieee754.py)

Реализует операции с числами с плавающей точкой по стандарту IEEE-754.

| Метод | Назначение | Входные данные | Выходные данные |
|-------|------------|----------------|-----------------|
| `float_to_binary()` | Преобразование в формат IEEE-754 | `num` - число с плавающей точкой | Массив из 32 битов |
| `binary_to_float()` | Преобразование из IEEE-754 | `binary_array` - массив битов | Число с плавающей точкой |
| `add()` | Сложение двух чисел | `num1`, `num2` - числа | Словарь с результатом |
| `subtract()` | Вычитание двух чисел | `num1`, `num2` - числа | Словарь с результатом |
| `multiply()` | Умножение двух чисел | `num1`, `num2` - числа | Словарь с результатом |
| `divide()` | Деление двух чисел | `num1`, `num2` - числа | Словарь с результатом |

### 4. Класс `BCDConverter` (bcd_converter.py)

Реализует операции в двоично-десятичном коде (8421 BCD).

| Метод | Назначение | Входные данные | Выходные данные |
|-------|------------|----------------|-----------------|
| `decimal_to_bcd()` | Преобразование в BCD | `num` - десятичное число | Массив битов в BCD |
| `bcd_to_decimal()` | Преобразование из BCD | `bcd_array` - массив битов | Десятичное число |
| `add()` | Сложение в BCD | `num1`, `num2` - числа | Словарь с результатом |
| `subtract()` | Вычитание в BCD | `num1`, `num2` - числа | Словарь с результатом |
| `bcd_array_to_string()` | Преобразование BCD в строку | `bcd_array` - массив битов | Строка вида "0001 0010 0011" |

### 5. Класс `ArithmeticOperations` (arithmetic_operations.py)

Главный класс, объединяющий все операции и добавляющий проверку результатов.

| Метод | Назначение | Входные данные | Выходные данные |
|-------|------------|----------------|-----------------|
| `convert_decimal_to_binary()` | Перевод числа во все три кода | `num` - число | Словарь с тремя кодами |
| `add_additional()` | Сложение в дополнительном коде | `num1`, `num2` | Результат с проверкой |
| `subtract_additional()` | Вычитание в дополнительном коде | `num1`, `num2` | Результат с проверкой |
| `multiply_direct()` | Умножение в прямом коде | `num1`, `num2` | Результат с проверкой |
| `divide_direct()` | Деление в прямом коде | `num1`, `num2`, `precision` | Результат с проверкой |
| `ieee_add()` | Сложение IEEE-754 | `num1`, `num2` | Результат с проверкой |
| `ieee_subtract()` | Вычитание IEEE-754 | `num1`, `num2` | Результат с проверкой |
| `ieee_multiply()` | Умножение IEEE-754 | `num1`, `num2` | Результат с проверкой |
| `ieee_divide()` | Деление IEEE-754 | `num1`, `num2` | Результат с проверкой |
| `bcd_add()` | Сложение BCD | `num1`, `num2` | Результат с проверкой |
| `bcd_subtract()` | Вычитание BCD | `num1`, `num2` | Результат с проверкой |
